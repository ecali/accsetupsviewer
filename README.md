# accsetupsviewer

Applicazione [Next.js](https://nextjs.org) con [TypeScript](https://www.typescriptlang.org/) e [Tailwind CSS](https://tailwindcss.com/).

## Avvio sviluppo

```bash
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000).

## Script disponibili

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
